{% trans "¿Qué es Neutrón?" %}

{% trans "Neutrón es un proyecto de investigación y una herramienta de ayuda a la escritura, traducción y revisión de textos viajeros." %}

{% trans "Sin ánimo de lucro" %}:

{% trans "Texto viajero" %}:

{% trans "haremos las cosas como creemos que se deben hacer." %}

{% trans "aquel cuyos autores y lectores son hablantes de variedades de español diferentes." %}

Neutrón: {% trans "el proyecto" %}

{% blocktrans %} Queremos identificar las diferencias que los hablantes de distintas variedades geográficas del español manifiestan en el uso del léxico. {% endblocktrans %}

{% blocktrans %} Como resultado, a través del proyecto obtendremos una base de datos de términos y acepciones etiquetados por regiones que arrojará información sobre el uso de la lengua en cada punto del mapa. {% endblocktrans %}

Neutrón: {% trans "la herramienta" %}

{% blocktrans %} Pondremos a disposición de profesionales y usuarios de la lengua española una herramienta que facilite la adaptación de los textos al español que su público objetivo realmente conoce y con el que se siente cómodo. {% endblocktrans %}

{% blocktrans %} Con esta herramienta contribuiremos a que los textos viajeros sean mejor comprendidos y valorados por sus lectores. {% endblocktrans %}

{% trans "Hablamos español en más de 20 países, ¡acerquémoslos!" %}

{% trans "¿Cómo funciona?" %}

{% trans "El proyecto Neutrón se alimentará de dos fuentes complementarias:" %}

{% trans "Informantes" %}: {% blocktrans %} hablantes de las distintas variedades geográficas del español recorrerán un glosario de términos y acepciones para etiquetarlos de acuerdo con su habla. {% endblocktrans %}

{% trans "Corpus" %}: {% blocktrans %} contrastaremos corpus procedentes de distintas variedades geográficas con el fin de identificar diferencias en la aparición de términos y en su colocación. {% endblocktrans %}

{% blocktrans %} La herramienta Neutrón, previamente instalada en el editor de textos, facilitará dos tipos de adaptaciones, en función de la distribución que quiera dársele al escrito: {% endblocktrans %}

{% trans "Localización" %}: {% blocktrans %} modificación del texto para que su léxico refleje el habla del lector de una región específica. {% endblocktrans %}

{% trans "Neutralización" %}: {% blocktrans %} modificación del texto para que su léxico sea reconocible y bien aceptado por la mayor cantidad posible de hispanohablantes en el mundo. {% endblocktrans %}

{% trans "Nuestros informantes: cuéntanos cómo es tu español" %}

{% blocktrans %} Buscamos hablantes de español con ilusión por reivindicar las peculiaridades de su habla y participar en un proyecto pionero. {% endblocktrans %}
